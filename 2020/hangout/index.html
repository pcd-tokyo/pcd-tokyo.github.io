Processing Community Hangout Japanプロセッシング・コミュニティ・ハングアウト・ジャパン

Processing Community Day（プロセッシング・コミュニティ・デー） は Processing のユーザーの交流イベントでアートとプログラミングのコミュニティの多様性を祝って世界各地で開催されており、現在はコロナウイルスの影響を受けてオンラインでの企画を行っています。そこで Processing 日本コミュニティとしても4-5月の「デイリーコーディングチャレンジ」や第1回ハングアウトを行ったほか、7月5日（日）の 19:30 - 21:30 に2回目のオンライン会を YouTube のライブ配信として計画しています。今回は、

  1. スケッチ投稿
  2. 30-min coding challenge の動画投稿
  3. お悩み相談

を募集しています。7月の 1. と 2. のお題は「七夕」です。

1. 「スケッチ投稿」

今回のハングアウトでは、七夕に関するみなさんのスケッチを募集します。七夕をテーマとした Processing または p5.js のスケッチであれば静止画・映像・インタラクションなどフォーマットは問いません。条件としては、OpenProcessing や p5.js Editor 、 Github (Gist) 等でオープン・ソースとして公開することです（アップロード方法が分からない方は @PCD_Tokyo へお気軽にご相談ください）。ダウンロード可能、あるいはブラウザで実行可能なリンクを @PCD_Tokyo にリプライとしてハッシュタグ「#pchj02」を添えて投稿してください。

当日はスケッチの質や技術などに関係なく、より多くの個性的な作品を紹介する予定です。

2. 「30-min coding challenge」

30-min coding challenge は事前に30分間でテーマに沿って一からコーディングしている様子を収録し、配信中にその動画を見ながらコメントをする企画です。最終的な作品だけでなく実際にコーディングしている様子をお伝えすることが目的で、30分内に完成させる必要はありません。5月31日には第0回として takawo・ひえだで配信を行いましたので下のアーカイブをご参考ください。

今回のテーマはスケッチと同様「七夕」です。30-min coding challenge の参加表明は @PCD_Tokyo にリプライかダイレクトメッセージからお願いします。手順などについて折り返しご連絡します。

3. 「お悩み相談」

「バグがあるけどどう直せばいいのか分からない」「クラスを使いたいけどどうやって始めたらいいのかわからない」「どこからインスピレーションを見つけたらいいの？」といったお悩みのある方、PCD Tokyo 2020 のお悩み相談室が帰ってきました！7月5日の配信または、それ以降の企画で Processing や p5.js ユーザが質問にお答えします。YouTube 上で配信を行うか、個別に Zoom 等で行うかなどフォーマットについては現在検討中です。興味のある方は @PCD_Tokyo にリプライかダイレクトメッセージしていただければ詳細について返信いたしますのでお気軽にご連絡ください！
